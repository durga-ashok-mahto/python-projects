# Python Mini Projects

This repository contains beginner Python projects created for practice and learning.

## Projects Included

1. Calculator
A simple calculator that performs basic operations like addition, subtraction, multiplication, and division.

2. To-Do List
A simple command-line to-do list program that allows users to add and view tasks.

3. Number Guessing Game
A small game where the user tries to guess a randomly generated number.

## Technologies Used

- Python

## How to Run the Programs

1. Download or clone the repository
2. Open the project folder
3. Run the Python files using the terminal

Example:

python calculator.py
